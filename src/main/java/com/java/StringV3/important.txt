My First Java Program

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

public – Access modifier (visible to all)
class – Blueprint for objects
main – Entry point of program
String[] args – Command-line arguments
System – Predefined class (java.lang)
out – Output stream connected to console
println – Prints text & moves to next line
static – Allows method to run without creating an object✏️.
